## The Scratch3 extension generator

Scratch3 is great, but writing extensions for it is not so easy. It could be too difficult for a no-pro coder and tedious to a pro. The purpose of this webapp is to make a scratch3 extension scaffold in less than 10 minutes. Hope you guys like it~
